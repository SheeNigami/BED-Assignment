
Prerequisites:

- NodeJS
- MySQL




Set-Up:

1) After downloading the project, run this command to install required node packages:


npm i



2) Next, run the queries in the file 'snapsell.sql' in your MySQL Terminal/Workbench.



3) Lastly, to set up the database connection, navigate to the file '/model/databaseConfig.js' and change the user and password info in .env.development


JWT_SECRET=SECRET-ECKS-dee
DB_HOST=localhost (CHANGE THIS)
DB_USER=BEDUser (CHANGE THIS)
DB_PASSWORD=BEDUser1!
DB_NAME=snapsell
PORT=3301



Start the Server:

Use:
npm run start-dev

to start the server
